      What this dashboard shows

      Trend color (1D): The dot color is based on the daily trend. Bull if Close > EMA50, EMA20 > EMA50 > EMA100, and EMA20 slope > +0.05%. Bear if Close < EMA50, EMA20 < EMA50 < EMA100, and EMA20 slope < −0.05%. Otherwise Neutral.

      PGL (momentum): L = (close − 24h low) / (24h high − 24h low). z = return vs prev close normalized by 24h range. Momentum label: Up if z ≥ 0 and L ≥ 0.60, Down if z ≤ 0 and L ≤ 0.40, Mid otherwise. PGL does not control color; it’s context.

      Each card: price and 24h change, 24h high/low, PGL L and z (intraday momentum), EMA20/50/100 and EMA20 slope% (trend). Links: open the exact symbol on TradingView and Binance.
